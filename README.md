Maxsteroids
===========

An imitation of a classic arcade game, created in order to learn Pygame. This was a project from my third week at Hacker School.

Runs on Python 2.x and Pygame (see requirements.txt)

Controls
--------
* Left+Right: Rotate ship
* Up: Accelerate
* Spacebar: Fire laser
* Enter: Hyperspace
